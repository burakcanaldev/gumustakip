# Gümüş Takip

Canlı gram gümüş fiyatı (Türkiye piyasası). Sade, hızlı, bağımsız.

**Site:** https://gumustakip.com

## Yapı

Tek dosya: `index.html`. Tüm CSS ve JS gömülü. Bağımlılık yok, build yok.

## Veri kaynağı

[truncgil finans API](https://finans.truncgil.com/) — TR piyasası fiyatları (gümüş, altın, döviz). 30 saniyede bir güncellenir.

## Deploy

GitHub Pages — `main` branch root'tan otomatik yayınlanır.
